CONTACT BOOK APPLICATION USING

DATA STRUCTURES

A contact book application is designed using Trie data structure.This application enables the users to add new contact,delete a contact,modify an existing contact.It also facilitates to search for a contact in the existing list.The use of Trie data structure to build this application is its ability to perform prefix based search.
